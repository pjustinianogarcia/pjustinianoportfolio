Projects

Letter: E-commerce Website

An original project simulating an eCommerce website where users can shop for handwritten letters. Built using HTML, CSS, and JavaScript, the application features automatic calculations, dynamic cart management, discount code functionality, and a responsive layout to enhance the user experience.

Pokédex

A web application simulating a Pokédex using HTML, CSS, JavaScript, and Bootstrap. The application loads data from an external API and allows users to view detailed information for each Pokémon through a responsive modal.

Meet App

A serverless progressive web application (PWA) developed with React using a test-driven development (TDD) approach. The application uses the Google Calendar API to fetch upcoming events.

MyFlix

MyFlix is a full-stack web application that enables users to explore a collection of movies, directors, and genres. Users can create accounts, view detailed movie information, and add their favorite movies to a list. The server-side is built with Node.js, Express.js, and MongoDB, while the client-side is developed using React.

About

My journey into web development began somewhat accidentally. While taking a Coursera course in Data Analytics, I gained substantial experience with Excel, but the part I enjoyed most was the brief introduction to SQL. Writing code, hitting "run," and seeing it work was incredibly enjoyable for me. That moment sparked my curiosity and led me to explore various programming languages and their capabilities. Web development stood out to me because of its unique blend of creativity and visual impact. This inspiration drove me to enroll in Career Foundry’s Full Stack Web Development program, where I immersed myself in coding and building interactive applications. I gained proficiency in HTML, CSS, JavaScript, and React while also gaining hands-on experience with MongoDB, GitHub, Node.js, Express.js, and Bootstrap. The program reinforced my ability to write clean, readable, and maintainable code. My passion for learning drives me to continuously grow and approach every challenge with optimism. Web development is the perfect field for me because it combines my creativity with my determination to build meaningful and impactful solutions.
