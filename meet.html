Meet App

This serverless progressive web application (PWA) was built using a test-driven development (TDD) technique. The app utilizes the Google Calendar API to fetch upcoming events and provides users with features to browse, filter, and view details about these events. The application supports offline functionality, enabling users to access previously viewed events without an internet connection.

Additional features include the ability to add an app shortcut to the home screen and visualize event data through interactive charts, such as displaying upcoming events by city. The app uses OAuth2 authentication with serverless functions hosted on AWS Lambda.

Key Features

  • Filter events by city.
  • Show/Hide event details.
  • Specify the number of events to display.
  • Use the app offline.
  • Add an app shortcut to the home screen.
  • Display charts visualizing event details.

Behavior-Driven Development (BDD) User Stories

The development process followed Behavior-Driven Development (BDD) principles to test various user stories, including:

  • As a user, I would like to filter events by city so that I can see the list of events taking place in a specific city.
  • As a user, I would like to show/hide event details so that I can view more or less information about an event.
  • As a user, I would like to specify the number of events I want to view in the app so that I can see more or fewer events in the list at once.
  • As a user, I would like to use the app offline so that I can access the events I viewed the last time I was online.
  • As a user, I would like to add the app shortcut to my home screen so that I can open the app faster.
  • As a user, I would like to see charts that visualize event details, such as upcoming events by city.

Data Visualization

CityEventsChart.js

This component visualizes event data as a scatter chart, mapping cities to event counts using the Recharts library. It dynamically processes data based on the events and allLocations props, providing an interactive visualization of event distributions.

EventGenresChart.js

This pie chart visualizes the distribution of events across programming genres, including React, JavaScript, Node, jQuery, and Angular. Powered by the Recharts library, the chart offers custom labels, dynamic updates, and vibrant colors to enhance data visualization.
